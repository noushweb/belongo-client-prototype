RTL implementation · for development handoff

How the mirror is built

Direction One attribute flips the layout

Set dir="rtl" lang="ar" on the root (or a region). Flexbox/grid order, text alignment and the sidebar position all mirror automatically — no duplicated layout.

Logical props No hard left/right

Use margin-inline, padding-inline, inset-inline-start instead of left/right so spacing mirrors. Audit the shared CSS for stray left/right before launch.

Numbers Keep prices & dates LTR

Wrap currency, totals, phone numbers and times in <bdi> (or dir="ltr") so “AED 952” and “600 Mbps” read correctly inside Arabic text.

Type An Arabic-first font

Pair a quality Arabic face (here, Noto Kufi Arabic) with the Latin display font. Line-height runs a touch taller for Arabic; the gold/navy tokens are unchanged.

Icons Mirror the directional ones

Flip chevrons, back arrows and progress steppers; leave symmetric icons (search, heart, the crescent) untouched. Use transform: scaleX(-1) only where direction has meaning.

Scope Proof, not full translation

This demo mirrors one page to prove the system holds. Translating all 36 screens is a content workstream — the layout, components and tokens are already RTL-ready.
